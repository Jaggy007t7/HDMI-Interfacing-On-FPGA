
# HDMI-Interfacing-On-FPGA

FPGA-based HDMI input and output interfacing project with real-time video processing and display implementation using Verilog HDL, Vivado, and Vitis.

---

# Projects Included

## 1. HDMI_Output_Using_Test_Pattern_Generator

- HDMI output at **720p resolution**
- Test pattern generated using **Verilog HDL**
- Real-time video display through HDMI

### Features
- 720p HDMI Output
- Verilog-based pattern generation
- Vivado IP Integrator design
